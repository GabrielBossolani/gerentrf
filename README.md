# GerenTRF

**GerenTRF** é um sistema simples de gerenciamento de tarefas, construído com Python e Flask.

## Funcionalidades
- ✅ Criar tarefas
- ✅ Listar tarefas
- ✅ Editar tarefas
- ✅ Excluir tarefas

## Tecnologias utilizadas
- Python
- Flask (microframework web)
